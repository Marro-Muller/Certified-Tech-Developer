
Objetivo

Trabalhar com mais de um arquivo html e css.

Instruções

Criar um menu suspenso e fixo no topo.
A página deve conter uma imagem que ocupe toda área de visualização.
A página deve conter produtos com botões estilizados.
Os botões devem conter links para outros documentos ou sites.
A página deve conter um ícone do whatsapp fixo na tela para contato com o vendedor.
