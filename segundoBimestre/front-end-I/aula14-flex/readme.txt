Agora que já vimos as diferentes formas de agrupar itens utilizando Flexbox, vamos construir uma interface de um site focado em streaming de vídeos (qualquer semelhança com um site real é mera coincidência).
Para isso, iremos criar um elemento pai, alguns elementos filhos onde deverão conter uma imagem para representar a capa do vídeo, um título e o número de visualizações.

Instruções:


Utilize o conceito que vimos nos slides/aula, onde para aplicarmos corretamente o Flexbox, devemos utilizá-lo no elemento pai, para que os elementos filhos sejam ajustados.
Lembre-se de utilizar a propriedade no elemento pai que nos permite dizer que os filhos podem ir para a linha de baixo caso eles não caibam na tela.
Lembre-se de definir um tamanho para os elementos filhos.
Como de costume e OBRIGATORIEDADE, vamos utilizar o CSS externo.
